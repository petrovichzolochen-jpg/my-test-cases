# ID: MATH_DIV_004_Проверка деления целых чисел с дробным результатом

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить деление двух целых чисел, результат которого будет дробным числом. Каждое число должно содержать не более 5 знаков.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**  
В поле "Результат" отображается корректное частное введённых чисел в виде десятичной дроби.
