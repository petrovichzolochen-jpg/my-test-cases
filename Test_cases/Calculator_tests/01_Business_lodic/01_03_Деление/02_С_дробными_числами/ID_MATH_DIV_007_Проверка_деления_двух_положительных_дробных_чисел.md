# ID: MATH_DIV_007_Проверка деления двух положительных дробных чисел

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить деление одного положительного дробного числа на другое положительное дробное число. Каждое число должно содержать не более 5 знаков, точка знаком не считается.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**  
В поле "Результат" отображается корректное частное введённых чисел.
