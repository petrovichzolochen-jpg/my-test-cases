# ID: MATH_DIV_008_Проверка деления двух отрицательных дробных чисел

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить деление одного отрицательного дробного числа на другое отрицательное дробное число. Каждое число должно содержать не более 5 знаков, точка знаком не считается.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**  
В поле "Результат" отображается корректное положительное частное введённых чисел.
