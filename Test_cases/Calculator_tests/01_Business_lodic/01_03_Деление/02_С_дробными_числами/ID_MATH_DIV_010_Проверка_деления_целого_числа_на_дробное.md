# ID: MATH_DIV_010_Проверка деления целого числа на дробное

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить деление целого числа на дробное число. Каждое число должно содержать не более 5 знаков, точка знаком не считается.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**
В поле "Результат" отображается корректное частное введённых чисел.
