# ID: MATH_DIV_011_Проверка деления, приводящего к периодической дроби

**Приоритет:** Средний

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить деление двух чисел, частное которых является бесконечной периодической десятичной дробью.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**  
В поле "Результат" отображается частное введённых чисел.
