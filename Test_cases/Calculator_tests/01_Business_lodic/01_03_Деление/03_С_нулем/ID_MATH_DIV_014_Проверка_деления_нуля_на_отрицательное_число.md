# ID: MATH_DIV_014_Проверка деления нуля на отрицательное число

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить деление нуля на отрицательное число. Число должно содержать не более 5 знаков.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**  
В поле "Результат" отображается частное введённых чисел.
