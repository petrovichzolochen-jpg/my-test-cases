# ID: MATH_DIV_016_Проверка деления нуля на ноль

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить деления нуля на ноль
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**  
В интерфейсе отображается сообщение об ошибке: "Деление на ноль невозможно".
