# ID: MATH_MUL_008_Проверка умножения целого и дробного числа

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить умножение целого числа и дробного числа. Каждое число должно содержать не более 5 знаков, для дробного числа точка знаком не считается.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**
В поле "Результат" отображается корректное произведение введённых чисел.
