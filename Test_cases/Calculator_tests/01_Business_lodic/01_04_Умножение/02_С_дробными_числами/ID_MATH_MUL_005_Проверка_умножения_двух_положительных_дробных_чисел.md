# ID: MATH_MUL_005_Проверка умножения двух положительных дробных чисел

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить умножение двух положительных дробных чисел. Каждое число должно содержать не более 5 знаков, точка знаком не считается.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**  
В поле "Результат" отображается корректное произведение введённых чисел.
