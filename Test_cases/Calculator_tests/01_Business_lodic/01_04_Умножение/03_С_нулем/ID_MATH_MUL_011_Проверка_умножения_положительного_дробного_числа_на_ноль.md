# ID: MATH_MUL_011_Проверка умножения положительного дробного числа на ноль

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить умножение положительного дробного числа на ноль. Число должно содержать не более 5 знаков, точка знаком не считается.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**
В поле "Результат" отображается число 0 (ноль).
