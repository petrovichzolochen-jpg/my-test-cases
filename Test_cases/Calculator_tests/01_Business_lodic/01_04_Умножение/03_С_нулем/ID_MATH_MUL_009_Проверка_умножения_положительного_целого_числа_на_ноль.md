# ID: MATH_MUL_009_Проверка умножения положительного целого числа на ноль

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить умножение положительного целого числа на ноль. Число должно содержать не более 5 знаков.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**  
В поле "Результат" отображается число 0 (ноль).
