# ID: MATH_006 Проверка операции сложения отрицательного однозначного числа и положительного однозначного числа
**Приоритет:** Средний  
**Шаги:**  
1. Открыть приложение "Калькулятор"  
2. В поле "Выражение" ввести: -8+6
3. Нажать кнопку "Вычислить"  
**Ожидаемый результат:**  
В поле "Результат" отображается число -2
