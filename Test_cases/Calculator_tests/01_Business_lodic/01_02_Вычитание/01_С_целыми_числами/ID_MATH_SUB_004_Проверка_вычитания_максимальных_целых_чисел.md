# ID: MATH_SUB_004_Проверка вычитания максимальных целых чисел

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить вычитание двух максимальных целых чисел. Каждое число должно содержать 5 знаков.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**  
В поле "Результат" отображается корректная разность введённых чисел.
