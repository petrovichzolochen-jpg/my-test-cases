# ID: MATH_SUB_005_Проверка вычитания минимальных целых чисел

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить вычитание двух целых чисел, состоящих из одного знака каждое.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**
В поле "Результат" отображается корректная разность введённых чисел.
