# ID: MATH_SUB_002_Проверка вычитания двух отрицательных целых чисел

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить вычитание двух отрицательных целых чисел. Каждое число должно содержать не более 5 знаков.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**  
В поле "Результат" отображается корректная разность введённых чисел.
