# ID: MATH_SUB_011_Проверка вычитания минимальных по длине дробных чисел

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить вычитание двух дробных чисел состоящих из двух знаков, точка знаком не считается.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**
В поле "Результат" отображается корректная разность введённых чисел.
