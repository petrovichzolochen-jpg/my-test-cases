# ID: MATH_SUB_007_Проверка вычитания двух отрицательных дробных чисел

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить вычитание двух отрицательных дробных чисел. Каждое число должно содержать не более 5 знаков, точка знаком не считается.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**  
В поле "Результат" отображается корректная разность введённых чисел.
