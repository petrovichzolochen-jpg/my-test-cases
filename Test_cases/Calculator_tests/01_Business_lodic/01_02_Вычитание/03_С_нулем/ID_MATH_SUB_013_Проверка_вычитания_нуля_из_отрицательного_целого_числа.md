# ID: MATH_SUB_013_Проверка вычитания нуля из отрицательного целого числа

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить вычитание нуля из отрицательного целого числа. Число должно содержать не более 5 знаков.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**  
В поле "Результат" отображается корректная разность введённых чисел.
