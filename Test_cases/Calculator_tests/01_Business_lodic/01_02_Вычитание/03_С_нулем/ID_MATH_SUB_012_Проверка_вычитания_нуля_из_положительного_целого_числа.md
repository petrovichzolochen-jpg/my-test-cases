# ID: MATH_SUB_012_Проверка вычитания нуля из положительного целого числа

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить вычитание нуля из положительного целого числа. Число должно содержать не более 5 знаков.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**  
В поле "Результат" отображается корректная разность введённых чисел.
