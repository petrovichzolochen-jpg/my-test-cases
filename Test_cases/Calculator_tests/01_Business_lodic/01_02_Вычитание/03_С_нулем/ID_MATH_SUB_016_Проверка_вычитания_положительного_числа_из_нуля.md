# ID: MATH_SUB_016_Проверка вычитания положительного числа из нуля

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить вычитание положительного числа из нуля. Число должно содержать не более 5 знаков.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**
В поле "Результат" отображается корректная разность введённых чисел.
