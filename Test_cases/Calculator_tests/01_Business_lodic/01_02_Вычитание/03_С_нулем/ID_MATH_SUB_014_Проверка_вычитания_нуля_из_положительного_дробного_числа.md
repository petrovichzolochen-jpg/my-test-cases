# ID: MATH_SUB_014_Проверка вычитания нуля из положительного дробного числа

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить вычитание нуля из положительного дробного числа . Число должно содержать не более 5 знаков, точка знаком не считается.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**
В поле "Результат" отображается корректная разность введённых чисел.
