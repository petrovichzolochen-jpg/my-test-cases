# ID: MATH_SUM_005_Проверка сложения двух минимальных целых чисел

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить сложение двух целых чисел, состоящих из одного знака каждое.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**
В поле "Результат" отображается корректная сумма введённых чисел.
