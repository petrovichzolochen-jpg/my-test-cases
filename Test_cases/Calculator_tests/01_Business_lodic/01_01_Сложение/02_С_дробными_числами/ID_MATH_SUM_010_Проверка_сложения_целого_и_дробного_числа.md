# ID: MATH_SUM_009_Проверка сложения целого и дробного числа

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить сложение целого числа и дробного числа. Каждое число должно содержать не более 5 знаков. Для дробного числа точка знаком не считается.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**  
В поле "Результат" отображается корректная сумма введённых чисел.
