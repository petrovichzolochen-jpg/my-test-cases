# ID: MATH_SUM_012_Проверка сложения минимальных по значению дробных чисел

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить сложение двух дробных чисел минимальных по значению.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**  
В поле "Результат" отображается корректная сумма введённых чисел.
