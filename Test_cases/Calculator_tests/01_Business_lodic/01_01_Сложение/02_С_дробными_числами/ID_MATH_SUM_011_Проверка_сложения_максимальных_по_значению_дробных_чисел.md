# ID: MATH_SUM_011_Проверка сложения максимальных по значению дробных чисел

**Приоритет:** Высокий

**Шаги:**
1. Открыть приложение "Калькулятор"
2. В поле "Выражение" выполнить сложение двух дробных чисел максимальных по значению.
3. Нажать кнопку "Вычислить"

**Ожидаемый результат:**  
В поле "Результат" отображается корректная сумма введённых чисел.
