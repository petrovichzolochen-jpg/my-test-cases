# ID: MATH_001: Проверка обработки деления на ноль
**Приоритет:** Критический  
**Шаги:**  
1. Открыть приложение "Калькулятор"  
2. В поле "Выражение" ввести: 10/0  
3. Нажать кнопку "Вычислить"
