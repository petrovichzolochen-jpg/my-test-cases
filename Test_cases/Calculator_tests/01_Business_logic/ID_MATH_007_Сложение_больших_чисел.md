# ID: MATH_007: Проверка операции сложения шестизначного и однозначного числа
**Приоритет:** Средний  
**Шаги:** 
1. Открыть приложение "Калькулятор"  
2. В поле "Выражение" ввести: 999999+1
3. Нажать кнопку "Вычислить"  
**Ожидаемый результат:**  
В поле "Результат" отображается число 1000000 (один миллион)
