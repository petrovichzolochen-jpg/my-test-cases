# ID: MATH 010 Проверка операции умножения больших чисел
Приоритет: средний  
## Шаги: 
1. Отправить POST-запрос на URL: http://sitename/calculate  
2. В теле запроса передать JSON: {"statement": "1000*1000"}  
### Ожидаемый результат:  
HTTP Status: 200 OK  
Тело ответа: {"answer": 1000000.0, "error": null}
